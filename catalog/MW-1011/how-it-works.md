# How it Works: Financial Planning & Analysis (FP&A) AI Worker

Builds financial models and delivers insights that inform business decisions This AI Worker amplifies your FP&A team's impact by handling the mechanics so they can focus on insights. It builds and maintains driver-based financial models, automatically updates forecasts as actuals flow in, generates management reports with variance commentary, and answers ad-hoc analysis requests in minutes instead of days. The worker connects financial metrics to operational KPIs, enabling true business partnership between finance and operations.

## Triggers

- Forecast cycle
- actual results availability
- analysis request
- model update requirement
- KPI threshold breach

## Agent Orchestration

1. **Data Integration Agent**
2. **Model Update Agent**
3. **Forecast Generation Agent**
4. **Variance Identification Agent**
5. **Commentary Agent**
6. **Report Generation Agent**
7. **Alert Agent**
