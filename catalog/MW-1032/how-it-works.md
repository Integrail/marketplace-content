# How it Works: External Candidate Sourcing AI Worker

Prospects passive candidates via LinkedIn and email with personalized outreach This AI Worker handles the most challenging—and highest-value—recruiting work: converting passive candidates into applicants. It works with recruiters to identify target candidates through LinkedIn searches (Recruiter or standard), then for each prospect, extracts comprehensive profile data, analyzes their background against job requirements, and crafts highly personalized LinkedIn messages and emails. Every outreach references specific aspects of the candidate's experience, dramatically improving response rates over generic templates.

## Triggers

- LinkedIn search URL
- saved candidate list
- role job description
- outreach preferences

## Agent Orchestration

1. **Search Execution Agent**
2. **Profile Extraction Agent**
3. **Background Analysis Agent**
4. **Message Writing Agent**
5. **Approval Agent**
6. **LinkedIn Send Agent**
7. **Email Send Agent**

## Outputs

- Personalized LinkedIn messages
- personalized emails
