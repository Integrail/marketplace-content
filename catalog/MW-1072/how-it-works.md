# How it Works: Virtual Data Room Automation AI Worker

Organizes, indexes, and extracts insights from VDR documents automatically This AI Worker transforms VDR review from a document-by-document slog into intelligent, searchable insight. It ingests all VDR contents, categorizes documents by type and relevance, extracts key data points into structured formats, identifies missing materials, and surfaces critical      Search Efficiency information proactively. The worker enables natural language queries across thousands of documents and tracks document versions and updates. Due diligence teams find answers in seconds instead of hours.

## Triggers

- VDR access granted
- document upload
- search query
- extraction request
- gap analysis request
- update notification

## Agent Orchestration

1. **Ingestion Agent**
2. **Classification Agent**
3. **OCR/Extraction Agent**
4. **Indexing Agent**
5. **Gap Analysis Agent**
6. **Search Agent**
7. **Alert Agent**
