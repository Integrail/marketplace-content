# How it Works: Use Case Discovery AI Worker

This AI Worker transforms discovery insights into tailored presentations. It analyzes call transcripts and rep inputs to identify which use cases resonate with each prospect, then creates a customized sales deck that highlights relevant solutions, success stories, and outcomes. Every prospect receives a presentation that speaks directly to their specific challenges and opportunities.

## Triggers

- Discovery call transcripts
- rep notes
- prospect industry/role
- deal context

## Agent Orchestration

1. **Transcript Analysis Agent**
2. **Use Case Matching Agent**
3. **Content Selection Agent**
4. **Deck Assembly Agent**
5. **Design Agent**

## Knowledge Sources

- Master slide library
- use case definitions
- customer success stories
- ROI data
- industry-specific content

## Outputs

- Customized sales deck
- tailored talk track
- relevant case studies
- personalized ROI projections
