# How it Works: Assessment AI Worker

This AI Worker adds objective data to your hiring decisions. It sends appropriate skills assessments based on role requirements, tracks completion, scores results, and integrates findings into candidate profiles. It can generate custom assessments for unique roles, flag concerning patterns, and provide hiring managers with data-driven insights beyond resume review.

## Triggers

- Candidate advances to assessment stage
- assessment completed
- reminder trigger
- results ready

## Agent Orchestration

1. **Assessment Selection Agent**
2. **Delivery Agent**
3. **Reminder Agent**
4. **Scoring Agent**
5. **Integration Agent**

## Knowledge Sources

- Role-assessment mapping
- scoring rubrics
- benchmark data
- question banks
- success profiles

## Outputs

- Assessment invitations
- completion tracking
- scored results
- candidate profile updates
- hiring manager summaries
